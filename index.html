M210 - Public Cloud für Anwendungen nutzen

Inhaltsverzeichnis

Public Cloud Serverless CI / CD Docker FTP

Public Cloud

Eine Public Cloud ist ein Cloud-Bereitstellungsmodell, bei dem sich Computing-Ressourcen im Besitz von Anbietern befinden, die diese Ressourcen verwalten und für mehrere Mandanten über das Internet bereitstellen.

Serverless

Der Begriff „Serverless" (serverlos) bezieht sich auf ein cloudnatives Entwicklungsmodell, bei dem Entwickler Anwendungen erstellen und ausführen können, ohne Server verwalten zu müssen.

CI / CD

CI/CD ist die kombinierte Praxis der Continuous Integration (CI) mit Continuous Delivery oder Continuous Deployment (CD). Der Zweck von CI/CD besteht darin, Entwicklungsteams die Möglichkeit zu geben, Codeänderungen häufiger und zuverlässiger durchzuführen.

Docker

Docker ist ein beliebtes Open-Source-Tool, das eine portable und konsistente Laufzeitumgebung für Softwareanwendungen bietet. Docker verwendet Container als isolierte Umgebungen im Benutzerraum, die auf Betriebssystemebene ausgeführt werden und das Dateisystem sowie die Systemressourcen gemeinsam nutzen.

FTP

FTP (File Transfer Protocol) ist ein Standard-Internetprotokoll, das, wie der Name schon sagt, zum Übertragen von Dateien zwischen Computern verwendet wird. FTP-Software verwendet ein Client-Server-Modell und erfordert daher zwei Teile, einen FTP-Client und einen FTP-Server.
